薅京东羊毛+青龙面板的安装

2021-10-17 2 min read # 日常

东东农场自动签到，自动签到领豆，自动给水果浇水·········

1.首先自备Docker

2.安装青龙面板

1.拉取青龙镜像

docker pull whyour/qinglong:latest

2.安装青龙面板

  • 服务器安装
docker run -dit \
   -v $PWD/ql/config:/ql/config \
   -v $PWD/ql/log:/ql/log \
   -v $PWD/ql/db:/ql/db \
   -v $PWD/ql/scripts:/ql/scripts \
   -p 5700:5700 \
   --name qinglong \
   --hostname qinglong \
   --restart always \
   whyour/qinglong:latest
  • N1盒子安装（旁路由设备）
docker run -dit \
   -v /mnt/mmcblk2p4/qinglong/config:/ql/config \
   -v /mnt/mmcblk2p4/qinglong/log:/ql/log \
   -v /mnt/mmcblk2p4/qinglong/db:/ql/db \
   -v /mnt/mmcblk2p4/qinglong/scripts:/ql/scripts \
   --net host \
   --name qinglong \
   --hostname qinglong \
   --restart always \
   whyour/qinglong:latest

安装完成后开启5700的端口访问，即可用IP:5700访问青龙面板了

3.安装必要依赖

docker exec -it qinglong bash -c "pip3 install requests"
docker exec -it qinglong bash -c "apk add python3 zlib-dev gcc jpeg-dev python3-dev musl-dev freetype-dev"
下边这个宠汪汪是必装的
docker exec -it qinglong bash -c "pnpm i png-js"
这个是宠汪汪活动的必要依赖，其他的依赖缺就装不缺就没啥。

4.拉库

库的解释：就是一个脚本的集合，一个库里面有很多脚本，脚本对应的京东的活动
