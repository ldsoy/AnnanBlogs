Clash For Windows启用虚拟网卡TUN模式

2021-09-30 2 min read # proxy

Clash for windows以下简称为cfw，这个软件的作者提到tun模式在windows上是支持最好的，所以在此说一下怎么打开虚拟网卡tun模式

  1. 先要下载支持tun的组件，去这里下https://www.wintun.net

    下载完是一个压缩包，里面有64的有x86的，找到跟你CFW对应版本的wintun.dll

  1. 把wintun.dll放进你们配置文件目录，点一下我标注的Open Folder就会弹出一个文件夹，那就是配置文件目录

  1. 启用Service Mode，并安装

    弄好后，Service Mode会变成雾蓝色的地球

    并且的你的配置文件的目录，会多一个Service的文件夹

  2. 到Settings配置Profile Mixin

    打开后，把下面的配置填进去

    mixin:
dns:
    enable: true
    default-nameserver:
      - 223.5.5.5
      - 1.1.1.1
    enhanced-mode: redir-host
    nameserver:
      - 223.5.5.5
      - 223.6.6.6
      - https://223.5.5.5/resolve
    fallback:
      - 1.1.1.1
      - 208.67.220.222:5353
      - https://dns.rubyfish.cn/dns-query
      - https://doh.opendns.com/dns-query
      - tls://1.1.1.1:853
    fallback-filter:
      geoip: true
tun:
    enable: true
    stack: gvisor
    dns-hijack:
      - 198.18.0.2:53
    macOS-auto-route: true
    macOS-auto-detect-interface: true
    ```

    就像下面这样

    ![](https://ldsoy.coding.net/p/PC/d/Img/git/raw/master/MDimg/20210930214503.png)

    填好点右下角保存就行了。

  3. 回主界面打开Mixin和system proxy就ok了

    现在就是开启了虚拟网卡模式

    当你不想使用TUN模式了的时候就把Mixin关了就行

补充

如果显示Clash网卡无法连接，请更换Mixin中的DNS，或者你可以删掉fallback部分
如下：

mixin:
   dns:
       enable: true
       default-nameserver:
         - 223.5.5.5
         - 1.1.1.1
       enhanced-mode: redir-host
       nameserver:
         - 223.5.5.5
         - 223.6.6.6
         - https://223.5.5.5/resolve
   tun:
       enable: true
       stack: gvisor
       dns-hijack:
         - 198.18.0.2:53
       macOS-auto-route: true
       macOS-auto-detect-interface: true
       ```
